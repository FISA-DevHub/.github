## 🚀 Getting Started

### Prerequisites
- ☕ jdk 17
- 📦 Maven 3.6+
- 🔧 IDE (IntelliJ IDEA or Eclipse recommended)

### Quick Start
```bash
# Clone the repository
git clone https://github.com/FISA-DevHub/java_study

# Navigate to project directory
cd java_study

# Build the project
mvn clean compile
```
